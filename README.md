# Snova LEADS — лендинг v2

## Что изменено
- Главный CTA теперь заглушка под будущий квиз.
- Кнопка в шапке ведет вниз к форме.
- Отступы между секциями уменьшены примерно на 30%.
- Заголовки блоков уменьшены.
- Первая строка двойных заголовков увеличена.
- Блоки «Почему со мной работают долго» и «Отзывы» собраны в общий двухколоночный блок ближе к первому превью.
- Иконки в карточках сделаны линейными в синем цвете через CSS.

## v11 правки
- Форма отправляет заявку напрямую в Telegram Bot API.
- Заявка отправляется на 2 Telegram аккаунта.
- После успешной отправки показывается уведомление.
- Если отправка не удалась, показывается ошибка.

Важно: токен бота сейчас находится в `script.js`. Для продакшена лучше перевыпустить токен и спрятать его через Cloudflare Worker.
